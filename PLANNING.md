# 무료/공영 주차장 찾기 - 앱인토스 미니앱 기획문서

---

## 1. 앱 개요

### 앱 이름
- **한글**: 주차해요
- **영문**: ParkFree

### 한줄 소개
> 내 주변 무료/공영 주차장을 실시간으로 찾고, 무료 주차 꿀팁까지 한눈에 확인해요.

### 타겟 사용자
| 구분 | 상세 |
|------|------|
| **핵심 타겟** | 20~50대 자가용 운전자 (토스 사용자 3,000만명 중 운전자) |
| **페인 포인트** | 주차장 찾기 어려움, 주차비 부담, 실시간 잔여석 정보 부재 |
| **사용 시나리오** | 외출 전/이동 중 목적지 근처 무료/저렴한 주차장 탐색 |
| **세부 타겟 1** | 주차비 절약이 필요한 직장인 (도심 출퇴근) |
| **세부 타겟 2** | 쇼핑/외출 시 무료 주차 혜택을 활용하고 싶은 가족 단위 사용자 |
| **세부 타겟 3** | 낯선 지역 방문 시 주차 정보가 필요한 여행/출장 사용자 |

## 2. 핵심 기능

### MVP 1차 (핵심 기능)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| **지도 기반 주차장 탐색** | 현재 위치 중심으로 주변 공영/무료 주차장을 지도에 마커로 표시해요 | P0 |
| **주차장 목록 보기** | 거리순/요금순으로 주차장 리스트를 정렬해서 보여줘요 | P0 |
| **주차장 상세 정보** | 주소, 운영시간, 요금, 총 주차면수, 전화번호, 길찾기 연동 정보를 제공해요 | P0 |
| **실시간 잔여석 표시** | 실시간 데이터가 연동된 주차장의 남은 자리 수를 보여줘요 (5분 주기 갱신) | P0 |

| **검색 기능** | 지역명, 주소, 랜드마크로 주차장을 검색할 수 있어요 | P1 |
| **즐겨찾기** | 자주 가는 주차장을 저장해서 빠르게 접근할 수 있어요 | P1 |

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| **주차 알림** | 즐겨찾기한 주차장의 잔여석이 일정 수 이하일 때 알림을 보내줘요 | P2 |
| **주차 이력 관리** | 주차한 위치와 시간을 기록해서 주차비 정산에 도움을 줘요 | P2 |
| **무료 주차 캘린더** | 공휴일, 주말 등 무료 개방 일정을 캘린더로 보여줘요 | P2 |
| **사용자 리뷰/제보** | 주차장 상태, 실제 무료 여부 등을 사용자가 제보할 수 있어요 | P2 |
| **경로 안내 연동** | 토스 앱 내에서 네이버/카카오 지도 길찾기로 바로 연결해요 | P2 |
| **주차 요금 계산기** | 예상 주차 시간을 입력하면 요금을 미리 계산해 줘요 | P3 |
| **민영 주차장 정보** | 공영 외 민영 주차장 정보도 추가로 제공해요 | P3 |
| **주차 꿀팁 커뮤니티** | 사용자들이 직접 무료 주차 팁을 공유할 수 있는 공간이에요 | P3 |

## 3. 화면 구성

### 화면 목록

#### 3-1. 홈 화면 (지도 뷰)
- **TDS Navigation**: 상단에 앱 이름 "주차해요" 표시, 검색 아이콘
- **지도 영역**: 전체 화면의 약 60%를 차지하는 지도 (현재 위치 중심)
- **주차장 마커**: 무료(초록), 공영(파랑 #3182F6), 유료(회색)로 구분
- **잔여석 배지**: 마커 위에 TDS Badge로 잔여석 수 표시 (여유: Green, 혼잡: Orange, 만차: Red)
- **하단 시트**: 주차장 리스트 미리보기 (TDS BottomSheet, 스와이프로 확장 가능)
  - TDS ListRow로 주차장 이름, 거리, 잔여석, 요금 간략 표시
- **플로팅 버튼**: 현재 위치로 이동, 목록 보기 전환

#### 3-2. 주차장 목록 화면
- **TDS Navigation**: 뒤로가기 + "주차장 목록" 타이틀
- **TDS Tab**: 전체 / 무료 / 공영 / 민영 탭으로 필터
- **정렬 옵션**: TDS SegmentedControl로 거리순 / 요금순 전환
- **주차장 리스트**: TDS ListRow 반복
  - 좌측: 주차장 유형 아이콘
  - 중앙: 주차장명, 주소, 운영시간
  - 우측: 거리(km), 잔여석 Badge, 요금
- **TDS BottomCTA**: "지도에서 보기" 버튼

#### 3-3. 주차장 상세 화면
- **TDS Navigation**: 뒤로가기 + 주차장 이름
- **상단 섹션 (TDS Top)**:
  - 주차장 이름, 유형 Badge (무료/공영/민영)
  - 실시간 잔여석: 큰 숫자로 강조 표시
  - 운영 상태 Badge (운영중: Green, 마감: Red)
- **정보 섹션**:
  - TDS ListRow: 주소 (지도 미니맵 포함)
  - TDS ListRow: 운영시간
  - TDS ListRow: 요금 정보 (기본/추가 요금)
  - TDS ListRow: 총 주차면수
  - TDS ListRow: 전화번호 (탭하면 전화 연결)
  - TDS ListRow: 주차장 유형 (노상/노외/부설)
- **무료 주차 조건** (해당 시): TDS Border로 감싼 안내 카드
- **TDS BottomCTA**: "길찾기" 버튼 (외부 지도 앱 연동)

#### 3-4. 무료 주차 꿀팁 화면
- **TDS Navigation**: 뒤로가기 + "무료 주차 꿀팁"
- **TDS Tab**: 마트/백화점 / 공공시설 / 병원/관공서 / 기타
- **카드 리스트**: 카테고리별 무료 주차 팁 카드
  - 장소명, 무료 주차 조건, 주의사항
  - 예시: "이마트 - 1만원 이상 구매 시 2시간 무료"
  - 예시: "주민센터 - 평일 18시 이후, 주말 종일 무료"
  - 예시: "공공도서관 - 이용자 3시간 무료"
- **인앱 광고 배너**: 화면 하단 AdMob 배너 광고 영역

#### 3-5. 검색 화면
- **TDS SearchField**: 상단 검색 입력 필드
- **최근 검색어**: TDS ListRow로 최근 검색 이력 표시
- **인기 검색어**: 많이 검색된 지역/장소 표시
- **검색 결과**: TDS ListRow로 주차장 리스트 표시

#### 3-6. 즐겨찾기 화면
- **TDS Navigation**: 뒤로가기 + "즐겨찾기"
- **즐겨찾기 리스트**: TDS ListRow로 저장된 주차장 표시
  - 실시간 잔여석 정보 함께 갱신
- **빈 상태**: 즐겨찾기가 없을 때 안내 일러스트 + "자주 가는 주차장을 저장해 보세요" 메시지

#### 3-7. 설정 화면
- **TDS Navigation**: 뒤로가기 + "설정"
- **TDS ListRow**: 알림 설정 (ON/OFF)
- **TDS ListRow**: 기본 검색 반경 설정 (500m ~ 5km)
- **TDS ListRow**: 데이터 출처 안내
- **TDS ListRow**: 서비스 이용약관
- **TDS ListRow**: 개인정보 처리방침
- **TDS ListRow**: 앱 버전 정보

##### 한국교통안전공단 주차정보 제공 API 상세

- **서비스 URL**: `http://apis.data.go.kr/B553881/Parking`
- **데이터 포맷**: JSON / XML
- **트래픽 제한**: 개발 계정 10,000건/일, 운영 계정 트래픽 증량 가능
- **인증 방식**: 서비스 키(API Key) 기반
- **데이터 규격**: ITS Korea 실시간 주차정보 수집/연계 규격 준용

##### 전국주차장정보 표준데이터 (15012896)

- **데이터 형식**: CSV / JSON / XML (오픈 API 자동 변환)
- **제공 항목**: 주차장 관리번호, 주차장명, 주차장 구분, 주차장 유형, 소재지 주소, 주차 구획수, 급지구분, 부제시행구분, 운영요일, 평일/토요일/공휴일 운영시간, 요금정보, 결제방법, 관리기관명, 전화번호, 위도, 경도 등 30+ 항목

- **활용 방식**: 공공데이터포털에서 OpenAPI로 자동 변환 제공, 활용 신청 후 사용

| [경기데이터드림 주차장 정보](https://data.gg.go.kr/) | 경기도 | 경기도 주차장 정보 및 실시간 주차 가능 |

#### 4-3. 지도 API

| API | 용도 |
|-----|------|
| **Kakao Maps JavaScript SDK** | 지도 렌더링, 마커 표시, 현재 위치 표시 |
| **Kakao Local API** | 키워드 검색, 주소 → 좌표 변환 (Geocoding) |

### 데이터 저장

| 저장소 | 용도 | 설명 |
|--------|------|------|
| **앱인토스 네이티브 스토리지** | 즐겨찾기, 설정 | `get/set/clear` API로 로컬 저장 |
| **앱인토스 서버 (mTLS)** | 사용자별 데이터 동기화 | 즐겨찾기, 검색 이력 등 서버 저장 (2차) |
| **클라이언트 캐시** | API 응답 캐싱 | 주차장 기본정보 로컬 캐시 (1시간 TTL) |
| **서버 DB (추후)** | 꿀팁 콘텐츠, 사용자 리뷰 | 자체 백엔드 서버 구축 시 활용 |

### 데이터 흐름

```
[공공데이터포털 API] ──→ [자체 프록시 서버 (mTLS)] ──→ [앱인토스 WebView 클라이언트]
                              │                              │
                              ├── 캐싱 (Redis)               ├── 네이티브 스토리지
                              └── 데이터 정제/가공            └── UI 렌더링 (TDS)
```

> **참고**: 공공데이터포털 API는 클라이언트에서 직접 호출하지 않고, 자체 프록시 서버를 통해 호출해요. 이유는 API 키 보호, 데이터 캐싱, 응답 포맷 통일, 그리고 앱인토스 mTLS 인증 요구사항 준수를 위해서예요.

## 5. 수익화 전략

### 수익 모델: IAA (In-App Advertising, 인앱 광고)

앱인토스 In-app Ads SDK (v1.11.0+)를 활용한 Google AdMob 광고를 주 수익원으로 해요.

#### 광고 유형 및 배치

| 광고 유형 | 배치 위치 | 노출 시점 | 예상 수익 |
|-----------|-----------|-----------|-----------|
| **배너 광고** | 꿀팁 화면 하단, 검색 결과 하단 | 화면 진입 시 상시 | 기본 수익 |
| **전면 광고 (Interstitial)** | 주차장 상세 → 길찾기 전환 시 | 1일 3회 제한 (UX 보호) | 중간 수익 |
| **보상형 광고 (Rewarded)** | 추가 꿀팁 잠금 해제, 광고 없이 검색 | 사용자 선택 시 | 높은 수익 |

#### 광고 정책 준수

- 앱인토스 인앱 광고 수수료: CBT 기간 15% (현재 0%)
- 과도한 광고로 인한 다크패턴 금지: 전면 광고는 1일 3회 이하로 제한
- 광고와 콘텐츠의 명확한 구분 (TDS Border 활용)
- 사용자가 광고를 닫을 수 있는 명확한 닫기 버튼 제공

#### 수익 시뮬레이션 (월 기준)

| 지표 | 수치 | 비고 |
|------|------|------|
| 예상 MAU | 50,000명 | 출시 3개월 후 기준 |
| 일일 활성 사용자 (DAU) | 10,000명 | MAU의 20% |
| 배너 광고 노출 (일) | 30,000회 | DAU x 평균 3페이지 |
| 전면 광고 노출 (일) | 5,000회 | DAU의 50% x 1회 |
| 보상형 광고 시청 (일) | 1,000회 | DAU의 10% |

#### 향후 수익화 확장 (2차)

## 8. 경쟁력 & 차별점

### 기존 경쟁 서비스 분석

| 서비스 | 특징 | 약점 |
|--------|------|------|
| **모두의주차장** | 공유 주차장 중심, 월 주차 특화 | 무료 주차 정보 부족, 별도 앱 설치 필요 |
| **카카오맵/네이버지도** | 주차장 표시 기능 내장 | 주차 전문 기능 아님, 무료 주차 팁 없음 |
| **서울주차정보** | 서울시 공식 앱, 실시간 정보 | 서울 한정, UX 노후화, 별도 앱 설치 |
| **파킹클라우드** | IoT 기반 실시간 정보 | 제휴 주차장 한정, 유료 서비스 중심 |

### 주차해요의 차별점

| 차별점 | 설명 |
|--------|------|
| **토스 앱 내 즉시 접근** | 별도 앱 설치 없이 토스 앱에서 바로 사용해요. 토스 3,000만 사용자 기반의 자연스러운 유입이 가능해요. |
| **무료 주차 꿀팁 특화** | 단순 주차장 검색을 넘어, 마트/백화점/공공시설별 무료 주차 조건을 체계적으로 정리해서 제공해요. 경쟁 서비스에 없는 고유 콘텐츠예요. |
| **전국 공영주차장 통합** | 서울뿐 아니라 전국 공영주차장 정보를 공공데이터포털 API로 통합 제공해요. 지역 제한이 없어요. |
| **실시간 + 정적 데이터 결합** | 실시간 잔여석 정보와 정적 기본 정보를 결합해서, 데이터가 없는 주차장도 기본 정보를 제공해요. |
| **TDS 기반 토스 네이티브 UX** | 토스 앱의 디자인 언어와 완전히 일치하는 UX로 사용자에게 이질감 없는 경험을 제공해요. |
| **가벼운 진입 장벽** | 회원가입, 로그인 불필요 (토스 로그인 자동). 앱을 열면 바로 주변 주차장이 보여요. |
| **광고 기반 완전 무료** | 사용자에게 모든 기능을 무료로 제공하고, 인앱 광고로 수익화해요. 유료 장벽이 없어요. |

### 핵심 경쟁력 요약

> **"토스 안에서 열면 바로 보이는, 전국 무료/공영 주차장 + 꿀팁"**
>
> 별도 설치 없는 즉시 접근성, 무료 주차 꿀팁이라는 고유 콘텐츠, 전국 공공데이터 통합의 3가지 축으로 기존 주차 앱과 명확히 차별화해요.
